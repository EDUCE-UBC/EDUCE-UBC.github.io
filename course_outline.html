Course outline

Data science

Introduction to data science

  • Introduction
    • Defines data science
    • Describes several uses and motivations for data science in the life sciences

Command line

Introduction to Unix

  • Introduction
    • Describes parts of the terminal / command prompt / shell
  • Download instructions
    • Provides instructions for the download and install terminals for Mac, Linux, and Windows
  • Unix tutorial and practice
    • Demonstrates basic functions like pwd, cd, ls, cp, mv…

Applications of command line

  • BLAST tutorial and practice
    • Complete BLAST of a large amplicon sequencing dataset via command line
  • Git tutorial and practice
    • Enact version control on a text file
  • Cloud computing tutorial and practice
    • Transfer files to and from the Google Cloud Platform (GCP)

R/RStudio

Introduction to R

  • Introduction
    • Introduces the R/RStudio program and community
  • Download instructions
    • Provides instructions for the download and install of R and RStudio
  • Base R tutorial and practice
    • Covers tabular data import and basic exploration
    • Includes creation of base R plots

Data manipulation and visualization in R

  • Employs the tidyverse packages to manipulate, clean, and create figures from tabular data

Intermediate data visualization in R

  • Utilizes data generated by students in laboratory courses and applies additional R packages, as needed
  • Customized for a given course

Statistics

Statistics basics

  • Introduces the Central Limit Theorem, p-hacking, and statistical tests relevant to the course material
  • Builds from t-tests to ANOVA to linear regression

Statistics in R

  • Applies statistical tests in R/RStudio
  • Includes data input, test completion, and interpretation of R outputs
  • Can utilize data generated by students in the course or our standard Saanich Inlet dataset

Capstone projects

Metagenomics analysis team project

  • Focuses on pipeline construction and biological interpretation of metagenomic sequence data from microbiomes

Microbiome analysis team project

  • Focuses on biological interpretation of amplicon sequence data from microbiomes
